This is a template Flask app with MongoDB.

## Dependencies

Commands to prepare environment

```
python3 -m venv venv/
source venv/bin/activate
pip install -r requirements.txt
export FLASK_APP=app.py
export FLASK_DEBUG=1

flask run
````
